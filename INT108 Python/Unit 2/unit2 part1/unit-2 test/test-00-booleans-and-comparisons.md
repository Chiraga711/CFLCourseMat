## Part A — Multiple Choice (5)

**Q1.** What is the result of `3 < 3`?
- A) `True`
- B) `False`
- C) `0`
- D) An error

**Q2.** What does the symbol `=` do in Python?
- A) Checks whether two values are equal
- B) Assigns a value to a variable
- C) Checks whether two values are not equal
- D) Nothing — it causes an error

**Q3.** What *type* of value does a comparison like `5 > 2` produce?
- A) `int` (a number)
- B) `str` (text)
- C) `bool` (a True/False value)
- D) It produces nothing

**Q4.** What is the result of `"cat" == "Cat"`?
- A) `True`
- B) `False`
- C) An error
- D) `"cat"`

**Q5.** What does this code print?
```python
result = 10 > 5
print(result)
```
- A) `10 > 5`
- B) `True`
- C) `False`
- D) `result`
